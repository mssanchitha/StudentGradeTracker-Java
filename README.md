# Student Grade Tracker (Java Swing GUI)

## Project Overview
The **Student Grade Tracker** is a Java‑based GUI application designed to input, manage, and analyze student grades efficiently.  
It calculates the **average**, **highest**, and **lowest** scores, and displays a summary report of all students using a simple and interactive interface.

---

## Aim
To develop a Java program that allows users to input student names and grades, store them dynamically, and generate a performance report using GUI components.

---

## Objectives
- Input and manage student grades through a user‑friendly interface.  
- Calculate **average**, **highest**, and **lowest** scores automatically.  
- Display a formatted summary report of all students.  
- Implement data storage using **ArrayList** for flexibility.  
- Provide a **Clear** option to reset data easily.  
- Enhance user experience with a clean GUI design.

---

## Technologies Used
| Component | Description |
|------------|--------------|

| **JDK Version** | 17 or higher |

---

## Algorithm / Workflow
1. Initialize two ArrayLists — one for student names and one for grades.  
2. Accept user input through text fields.  
3. Validate numeric input for grades.  
4. Store each entry in the respective ArrayLists.  
5. On “Add Student” click → entry is stored and displayed.  
6. On “Show Report” click → calculate average, highest, and lowest scores, then display results.  
7. On “Clear” click → reset all fields and output area.
